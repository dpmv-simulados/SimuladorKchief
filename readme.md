# Simulador de Provas - K-Chief 700

Simulador de provas interativo para preparação no sistema K-Chief 700, baseado em 100 questões do treinamento oficial.

## 🚀 Funcionalidades

- ✅ Testes com número personalizado de questões (1-100)
- ✅ Seleção aleatória de questões de diferentes assuntos
- ✅ Sistema de auto-avanço para respostas corretas (5 segundos)
- ✅ Bloqueio de mudança de opção após erro
- ✅ Explicação detalhada para respostas erradas
- ✅ Revisão apenas das questões erradas
- ✅ Histórico completo de resultados com nome, data e hora
- ✅ Sistema de aprovação (70% de acertos)
- ✅ Persistência de dados com localStorage

## 📋 Pré-requisitos

- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- JavaScript habilitado
